
# MODERN-TERMINAL-AI 1.18.31
🖥️ ModernCmd: the tool I always wanted to have

For a long time, I wanted to create a tool that brought together, in one place, many of the functions I regularly use as a Windows systems administrator.

That project is now a reality: ModernCmd is fully functional, available in both Spanish and English, and I honestly believe the result is excellent.

The application allows me to work with CMD and PowerShell through a more modern interface, review a persistent command history, access network and administration utilities, customize its appearance, and use artificial intelligence to generate or understand commands through natural language.

In the animated image, I show how the AI feature works: I simply describe what I need to do, and the tool helps turn it into a useful command.

ModernCmd is not intended to replace technical knowledge, but to make daily work easier, save time, and provide a more organized and accessible way to use the Windows command line.

A personal project I had wanted to build for a long time and that I can finally share.
